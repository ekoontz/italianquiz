Natural Language Sentence Generation

Context-Free Grammars

  • Sentence → NounPhrase VerbPhrase
  • VerbPhrase → Verb NounPhrase
  • NounPhrase → Determiner Noun

Problem: Agreement

  • La donna (the woman)
  • * Il donna
  • FemNounPhrase → FemDeterminer FemNoun
  • MascNounPhrase → MascDeterminer MascNoun

Unification Grammars

Structure Sharing

  • NounPhrase[gender=x] → Determiner[gender=x] Noun[gender=x]

Sentence generation = grammar + lexicon + unification

Clojure maps and refs

Map: set of key-value pairs, with nesting: trees

Maps + Refs = Directed Acyclic Graphs
